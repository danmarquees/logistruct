# 🚀 LogiStruct

**LogiStruct** é um sistema de gestão logística focado em insumos, construído para oferecer latência ultra-baixa e segurança de dados absoluta.

## 🛠 Tech Stack
- **Backend:** Go (concorrência e API)
- **Core Logic:** Rust (cálculos tributários e integridade)
- **Database:** PostgreSQL + PL/pgSQL
- **Mobile:** Kotlin Multiplatform (KMP)

## 🏗 Estrutura do Projeto
- `/cmd`: Pontos de entrada da aplicação Go.
- `/internal`: Lógica de negócio privada (Go).
- `/rust-core`: Biblioteca em Rust para cálculos críticos.
- `/proto`: Definições gRPC/Protobuf.
- `/web`: Dashboard administrativo.
